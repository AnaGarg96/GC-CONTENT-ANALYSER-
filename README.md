## Description 
  This is a calculator designed to analyse the percentage and amount of base pairs g and c present 

## Requirements 
  python 3.xx

## How To Run 
1. download the repository
2. open terminal in the file project 
3. run : gc calculator.py

## Input 
  base pairs - a , t , g , c 

## Output 
  calculated gc percentages
